# Productivity

通用工作流工具，不限于代码。

- **[caveman](./caveman/SKILL.md)** — 超压缩通信模式。通过去掉填充词，在保持完整技术准确性的同时，token 用量减少约 75%。
- **[grill-me](./grill-me/SKILL.md)** — 就一项计划或设计被无情地追问，直到决策树的每个分支都被理清。
- **[handoff](./handoff/SKILL.md)** — 将当前对话压缩成一份交接文档，让另一个智能体可以继续工作。
- **[write-a-skill](./write-a-skill/SKILL.md)** — 以正确的结构、渐进式披露和捆绑资源创建新技能。
