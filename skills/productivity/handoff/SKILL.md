---
name: handoff
description: 将当前对话压缩成一份交接文档，以便另一位智能体接手。
argument-hint: "下一会话将用于做什么？"
---

撰写一份交接文档，总结当前对话，以便新的智能体继续工作。保存到用户操作系统的临时目录中，而非当前工作区。

在文档中加入“建议使用的技能”部分，列出该智能体应调用的技能。

不要重复已在其他工件（PRD、计划、ADR、issue、提交、diff）中记录的内容。改而通过路径或 URL 引用它们。

隐去所有敏感信息，例如 API 密钥、密码或个人身份信息。

如果用户传入了参数，请将其视为对下一会话重点的描述，并据此调整文档内容。
